# GCP icons (populate manually)

Google's official Cloud Icons require accepting their asset license on
download — can't be auto-fetched or redistributed in this public repo.

1. Download from: https://cloud.google.com/icons
2. Extract the SVGs you want into this folder (subfolders OK, e.g. gcp/compute/vm.svg).
3. search_official_icon will pick them up automatically (domain="gcp").
