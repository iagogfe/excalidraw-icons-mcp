# Network / Cisco icons (populate manually)

Cisco's official Network Topology Icons require a cisco.com account to
download — can't be auto-fetched or redistributed in this public repo.

1. Download from: https://www.cisco.com/c/en/us/about/brand-center/network-topology-icons.html
2. Extract the SVGs/PNGs you want into this folder.
3. search_official_icon will pick them up automatically (domain="network").
