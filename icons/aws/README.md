# AWS icons (populate manually)

AWS's official Architecture Icons require accepting their asset license on
download — can't be auto-fetched or redistributed in this public repo.

1. Download the "Asset Package" zip: https://aws.amazon.com/architecture/icons/
2. Extract the SVGs you want into this folder (subfolders OK, e.g. aws/compute/ec2.svg).
3. search_official_icon will pick them up automatically (domain="aws").
