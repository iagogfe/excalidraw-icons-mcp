# Azure icons (populate manually)

Microsoft's official Azure Architecture Icons require accepting their asset
license on download — can't be auto-fetched or redistributed in this public repo.

1. Download the "Asset Package" zip: https://learn.microsoft.com/en-us/azure/architecture/icons/
2. Extract the SVGs you want into this folder (subfolders OK, e.g. azure/compute/vm.svg).
3. search_official_icon will pick them up automatically (domain="azure").
